# UP Yearning Society: IG Notes Generator

A React web application that uses a specialized Gemini-powered AI agent to match user moods with curated and dynamically discovered OPM/indie music for shareable Instagram Notes.

## Live Demo

**🚀 View the live application here:** [https://up-yearning-society-ig-notes.web.app](https://up-yearning-society-ig-notes.web.app)

## Features

- **Local Music Library Matching**: Falls back to a curated local library of OPM/indie music.
- **AI Discovery**: Uses Gemini 2.5 Flash to dynamically discover matching songs based on "hugot" culture.
- **Instagram-ready**: Mobile-first design perfect for IG notes formatting.
- **UP Branding**: Follows the UP Maroon, Forest Green, and Gold color palette.

## Tech Stack

- React 19
- TypeScript
- Vite
- Google Generative AI SDK (`@google/generative-ai`)
- Firebase Hosting
- Lucide React (Icons)
